## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- XAMPP

<!-- ```sh
 gem install rails
``` -->

### Setup

Clone this repository inside xampp/htdocs:

```sh
  git clone https://github.com/MAleemH/Leave-And-Attendance-Record-System.git
```

### Install

Install this project with:

- Create a database with exact name of .db file included in db folder
- Import .db file provided in db folder into newly created database

<!-- ```sh
  bundle install
``` -->

### Usage

To run the project:

- Open your desired web browser
- Type 'localhost/Leave-And-Attendance-Record-System' in URL and press enter

<!-- ```sh
